## Modification Examples

### Example 1: Push-Up with Wrist Pain

**Original Exercise:** pushup (standard hand placement)

**Severity Flag:** modify

**Modification Options:**
1. Use fists instead of flat palms (neutral wrist)
2. Use push-up bars (elevated grip)
3. Substitute: dumbbell_chest_press (neutral grip)
4. Substitute: band_pull_apart (no wrist load)

**Recommended Path:** Band work → Dumbbell work → Progress strength

---

### Example 2: Squat with Knee Injury

**Original Exercise:** barbell_back_squat

**Severity Flag:** avoid

**Modification Options:**
1. Reduce depth (partial squat) — NOT RECOMMENDED
2. Switch to: band_lateral_walk (activation only)
3. Switch to: glute_bridge (no knee bend stress)
4. Progress to: goblet_squat (controlled, light load)

**Recommended Path:** Band walk → Glute bridge → Goblet squat → Progress as tolerated

---

### Example 3: Turkish Get-Up with Shoulder Injury

**Original Exercise:** kettlebell_turkish_getup

**Severity Flag:** avoid

**Modification Options:**
1. Use light weight only (not recommended for this exercise)
2. Switch to: dead_hang (shoulder stability work)
3. Switch to: band_face_pull (posterior shoulder focus)
4. Full alternative: Build shoulder strength first via band work

**Recommended Path:** Band face pull → Dead hang → Return to TGU with very light weight
